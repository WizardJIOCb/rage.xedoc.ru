# 💀 RAGE ARENA — Убойный онлайн чат

Супер весёлый реалтайм чат, где **ругательства = урон**.

## Как играть

1. Запусти приложение
2. Открой несколько вкладок браузера (или попроси друзей зайти)
3. Введи уникальный ник и войди в арену
4. Пиши сообщения вида:
   - `@Громила ты сука!`
   - `Эй @Боец пиздец какой ты тупой`
5. Каждый раз когда сообщение содержит **мат + @упоминание** — цель теряет 14-42 HP

## Механики

- **HP** — здоровье. Падает от атак
- **Рейтинг** — поднимается когда ты успешно атакуешь (+9), падает когда тебя убивают (-55)
- **Смерть** — когда HP доходит до 0:
  - Игрок **замолкает на 60 секунд**
  - Рейтинг сильно снижается
  - После таймера — воскресает с полным HP
- Боты активно троллят всех. Открывай 3-4 вкладки, чтобы устроить настоящий бедлам

## Запуск

```bash
npm install
npm start
```

Открой http://localhost:3000

## Управление

- Клик по игроку в списке слева → вставит `@Имя`
- Двойной клик → моментальная атака (быстрый роаст)
- Отправляй любые сообщения. Только маты + @ = урон

## Список ругательств (примеры)

бля, сука, пиздец, хуй, ебать, мудак, дебил, идиот, чмо, и другие + немного английских.
